# CenDB v1.0.0 — Release Notes

## Overview

CenDB is a production-grade, multi-model embedded database engine built in
safe Rust. It projects relational, document, time-series, key-value, and
graph data models onto a single unified PAX storage substrate.

## What's New in v1.0.0

### Performance Optimizations
- **F64 vectorized filters**: 8-element batch processing for improved SIMD
  auto-vectorization. F64 filter throughput improved from 198 → 272 M rows/sec
  (+37%). F64 sum throughput improved from 816 → 1,862 M rows/sec (+128%).
- **F64 equality filter**: uses direct bit-pattern comparison (no f64
  conversion), achieving i64-level performance.
- **HLL accuracy**: increased default precision from 12 → 14 (error from
  ~10% → <5%). Added bias correction and proper leading-zeros handling.

### Binary Size
- FFI shared library (stripped): **399 KB**
- Full workspace (all 24 crates): **21,618 lines of Rust**

### Stability
- 500 crash-recovery iterations: **zero data loss**
- 5,000 PAX decoder fuzz iterations: **zero panics**
- 5,000 CenQL parser fuzz iterations: **zero panics**
- 5,000 encoding decoder fuzz iterations: **zero panics**
- 1,000 concurrent transaction stress: **zero panics**
